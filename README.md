# Scanbot Barcode Native Wrapper SDK for iOS

Swift Package Manager repository for the iOS [Scanbot Barcode SDK Native Wrapper](https://scanbot.io/sdk).

This SPM package is not intended for direct use. It is used internally by the Scanbot SDK to provide the native wrapper functionality for the iOS platform.
If you want to use the Scanbot Barcode SDK in your iOS project, please refer to the [Scanbot SDK documentation](https://docs.scanbot.io/ios/barcode-scanner-sdk/introduction/) for proper integration instructions.
